File::writeLine()

File::writeLine() -- writes a single line to a file

Synopsis

require_once 'File.php';

mixed File::writeLine (string $filename, string $line [, string $mode = FILE_MODE_APPEND [, string $crlf = "\n" [, mixed $lock = FALSE]]])

Description

Parameter

Return value

Throws

Note

This function can be called statically.

See

Example

Example 38-1. Using File::writeLine()

<?php
require_once 'File.php';

$e = File::writeLine('test.txt', str_repeat("0123456789", 1000));

if (PEAR::isError($e)) {
    echo 'Could not write to file : ' . $e->getMessage();
} else {
    echo "Successfully wrote to file test.txt\n";
}  
?>
