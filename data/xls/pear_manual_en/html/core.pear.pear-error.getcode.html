PEAR_Error::getCode()

PEAR_Error::getCode() -- get error code

Synopsis

require_once 'PEAR.php';

integer PEAR_Error::getCode ()

Description

Returns the error code coming with the error object.

Return value

integer - error number

Note

This function can not be called statically.
