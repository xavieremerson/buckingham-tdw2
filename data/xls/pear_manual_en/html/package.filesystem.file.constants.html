Constants

Constants -- Predefined Constants

FILE_DEFAULT_READSIZE

The default number of bytes to read from a file.

Used in

  • File::read()

  • File::readAll()

FILE_MODE_READ

Read-only mode for opened files

Used in

  • File::read()

  • File::readLine()

  • File::rewind()

  • File::close()

  • File::unlock()

FILE_MODE_WRITE

Using this mode, opened files will be truncated first and then new data will be written to them.

Used in

  • File::write()

  • File::writeChar()

  • File::writeLine()

  • File::rewind()

  • File::close()

  • File::unlock()

FILE_MODE_APPEND

Using this mode, new data will be appended to the end of opened files.

Used in

  • File::write()

  • File::writeChar()

  • File::writeLine()

  • File::rewind()

  • File::close()

  • File::unlock()

FILE_LOCK_SHARED

Shared (read) locking mode

Used in

  • File::read()

  • File::readChar()

  • File::readLine()

  • File::readAll()

  • File::write()

  • File::writeChar()

  • File::writeLine()

FILE_LOCK_EXCLUSIVE

Exclusive (write) locking mode

Used in

  • File::read()

  • File::readChar()

  • File::readLine()

  • File::readAll()

  • File::write()

  • File::writeChar()

  • File::writeLine()
