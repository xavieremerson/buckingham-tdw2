Manual installation

Warning

We removed this section, because, today, manually installing a package requires a deeper understanding of the way how packages are organized and what happens during the installation process. You should read the section about the package.xml in the Developers Guide (Chapter 15 and Chapter 16), if you really want install a package without the PEAR installer.

If you want to install PEAR on a remote host without shell access, you should look into the Section called Installation of a local PEAR copy on a shared host.
