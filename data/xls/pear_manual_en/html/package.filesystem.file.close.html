File::close()

File::close() -- closes an open file pointer

Synopsis

require_once 'File.php';

mixed File::close (string $filename, string $mode)

Description

Parameter

Return value

Throws

Note

This function can be called statically.

See

Example

Example 38-1. Using File::close()

<?php
require_once 'File.php';

$e = File::close('test.txt', FILE_MODE_WRITE);

if (PEAR::isError($e)) {
    echo 'Could not close file : ' . $e->getMessage();
} else {
    echo "Successfully closed file test.txt\n";
}
?>
