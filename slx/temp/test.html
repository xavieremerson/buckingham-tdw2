Overview

Tigra Tree Menu PRO is cross-browser JavaScript Tree Menu navigation component for web sites and web applications. High performance, efficient resources utilization and compact configuration files of the component make it possible to use it with huge hierarchies. Very little or no web development experience is required to start using the script. Powerfull features are available for advanced programmers.

Features
Even if half of the items below sound greek to you, sooner or later you'll find them all very useful. Being web developers ourselves and communicating with other programmers for years we've included in the list and implemented all most demanded features .
  • All styles (fonts, colors, distances, decorations) are defined with CSS independently for each state of the item. Settings are applied globally for the tree instance and can be overridden locally for each item.
  • Unlimited number of tree menus on single page all independently configurable
  • Unlimited hierarchy depth, unlimited number of children the node can have
  • Supports frames and windows targeting (other frame, new window, parent frame). The target can be defined globally for the tree instance and can be overridden locally for each item.
  • Pure relative positioning. Put tree menu code anywhere in the HTML document and see the menu at exactly the same place on the page
  • Rollover, on open and on select effects can be implemented
  • Optional multiple node selection (NEW)
  • Any tree node can be selected initially (NEW)
  • Any extra HTML attributes including more event handlers can be added to tree nodes (NEW)
  • Items' captions can contain any html (tables, pictures etc.)
  • Status bar messages and tool tips can be independently defined for each tree item
  • Efficient item search mechanisms (UPDATED)
  • Custom tree index to access it from third party scripts (NEW)
  • Tree state is kept across the pages and can be reset or redefined at any moment (UPDATED)
  • Events Model allows you to hook your code on tree menu events such as node open/close, tree show and others. You can even cancel the default functionality and replace it with your own.
  • Open Item Manipulation API makes it possible to control the state of the tree with external javascript code.
  • All popular browsers supported (compatibility table)
  • Low weight and high performance. Thousands of items are smoothly manipulated even on old computers.
  • Optimized for dynamic content: configuration files formats are simple and space efficient. Pull and visualize any hierarchical data from database, file system or whatever.
  • Tons of support materials and friendly competent staff ready to help when you have a question
  • PHP and ASP samples for dynamic tree generation from the database or file system
  • Reasonable price (only $29) and discounts. Just compare to what our competitors ask for.
