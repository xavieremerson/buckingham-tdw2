Where to start

First of all look inside /examples/grid/ folder. It contains several simple examples how to use grid with different data sources.

Try to read 'How-to' and some other sections in the inline documentation:

  • Grid How-To
  • Overview
  • General FAQ

There are several other sites which provide advanced examples and other hints related to ActiveWidgets library:

  • User examples

And finally you can look at complete ActiveWidgets source code inside /source folder.

Installation

Just copy /runtime folder to your webserver.

Debugging and Source Code

ActiveWidgets Grid library is provided both in compressed format (/runtime folder) and in full source code form (/source folder).

You can use full source version for debugging your code - change references to grid.js and grid.css files from /runtime to /source folder.

Community Support

The best place to find more information about the grid library is ActiveWidgets support forum. You may find that someone already had a similar problem and found a solution and we would be very grateful if you share your own experience with us and other developers.

Browser versions

ActiveWidgets grid works in:

  • Internet Explorer 5.5, 6.0
  • Netscape 7.1, 7.2
  • Mozilla 1.4 - 1.8
  • Firefox 0.6 - 1.5

Does not work in Opera, Konqueror, Safari.
