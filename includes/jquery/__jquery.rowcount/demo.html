Enable row count on example #1

Code:

$("table#rowCountTable").rowCount();

Example #2:

  Names
5. Daryl
6. David
7. Davy

Remove row on example #2

Code:

$("table#rowCountTable").rowCount({startAt:5, column:2});

There are only 2 options:

  • startAt - lets you start the next count at a specific number
  • column - lets you display the row count at a specific column

It also creates 2 css classes:

  • rowCount - for the column headers
  • rowNumber - for the cells

Apply as many times on an existing table to have it recount the rows - as shown on example #2
