Axis.setLinearScale

Usage

setLinearScale(lowerLimit, upperLimit[, majorTickInc[, minorTickInc ]])

Description

Set the axis to use the given linear scale.

Arguments

ArgumentDefaultDescription
lowerLimit(Mandatory)The lower bound of the axis.
upperLimit(Mandatory)The upper bound of the axis.
majorTickInc0Add major ticks to the axis, where the major ticks are separated by majorTickInc in value. Each major tick will have an associated text label for the value if the axis at the tick.

The default value of 0 means the major ticks will be automatically determined. In this case, the lowerLimit and upperLimit may be automatically adjusted to align with the ticks.

Use NoValue to disable major ticks.
minorTickInc0Add minor ticks to the axis, where the minor ticks are separated by minorTickInc in value. The default value of 0 means no minor tick is used.

Return Value

None
